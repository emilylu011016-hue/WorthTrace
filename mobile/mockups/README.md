# 钱迹WorthTrace 手机端 V2 Mock

更新时间：2026-06-12

## 这个 mock 是什么

这是手机端新版方向设计稿，不接数据库，不影响正式 App。

打开方式：

`docs/mobile-mockups/index.html`

## 新产品方向

手机端不再只是桌面 App 的缩小版。

新的手机端重点是：

1. 首页随身看板
2. 实时记账
3. 资产 Wallet
4. 月底轻更新

## 关键逻辑变化

以前：

月底导入账单 → 确认收支 → 资产录入 → 信用卡调整 → 生成月报

现在：

日常手机记账 → 月底只补资产和信用卡 → 生成月报

## 推荐技术路线

第一阶段建议先做手机 Web / PWA。

原因：

- 和当前桌面 Web 技术最接近
- 可以复用现有前端逻辑
- 成本比原生 App 和小程序低
- 后续可以再包装成真正的手机 App

## 暂不建议先做小程序

原因：

- 小程序不能直接读取本机 SQLite
- 数据同步和账号体系要重做
- 如果给朋友使用，需要云端数据库和部署成本
